## About the Project
This project is a simple language translation tool developed as part of an AI internship task.  
It allows users to enter text, choose a source language and a target language, and get the translated result instantly.

The main aim of this project is to understand how language translation APIs work and how they can be integrated into a simple user interface.

## Features
- User can enter any text for translation
- Option to select source and target languages
- Translates text using an API
- Displays the translated output clearly
- Simple and easy-to-use interface

## Technologies Used
- Python
- Streamlit
- Googletrans library
