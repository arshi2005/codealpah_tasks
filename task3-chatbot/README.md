# Simple AI Chatbot

## About the Project
This project is a simple AI chatbot developed as part of Task 3 of the AI internship.
The chatbot can respond to basic user inputs and maintains a chat-style interface.

The goal of this task is to understand how conversational logic works and how chat
applications can be built using Python.

## Features
- Chat-style user interface
- Responds to basic user questions
- Maintains conversation history
- Simple and easy to use

## Technologies Used
- Python
- Streamlit

## How to Run
1. Install the required library:
   pip install -r requirements.txt
2. Run the chatbot:
   streamlit run chatbot_app.py
